# 🚗 LocFácil

**LocFácil** é um sistema web desenvolvido em **Django** para gestão de locação de veículos, com foco em motoristas de aplicativo.  
A aplicação permite cadastrar motoristas, veículos, registrar locações e acompanhar o lucro de cada operação.

---

## 📋 Funcionalidades

- Cadastro de **motoristas**  
- Cadastro de **veículos**  
- Controle de **locações ativas e finalizadas**  
- Cálculo de **lucro estimado** por período  
- Interface administrativa (Django Admin)  
- Sistema preparado para expansão da frota

---

## 🛠️ Tecnologias utilizadas

- [Python 3.x](https://www.python.org/)
- [Django](https://www.djangoproject.com/)
- [SQLite](https://www.sqlite.org/) (banco padrão para desenvolvimento)
- [python-decouple](https://pypi.org/project/python-decouple/) — gerenciamento de variáveis de ambiente
- [Bootstrap 5](https://getbootstrap.com/) — estilização das páginas (opcional)
